# inu

A simple "cat" clone written in Rust.
Should not be used at all and was just a little project to learn Rust a bit.

# Usage

```
inu [OPTIONS] filename1 ...
```

Possible options:
```
-n		enumerate the beginning of each line
-e		print $ behind each line
```
